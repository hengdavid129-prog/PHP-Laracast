notes/index.php
    index will show all of a resource. Give me all notes.

notes/show.php
    show will show me a specific note.

notes/cteate.php
    create will show me a form to create a new note.


notes/store.php
    store will be what we hit after you submit that create form. So this is responsible for storing the note, for persisting the notes.

notes/edit.php
    edit show a form to edit a note.

notes/update.php
    update is where that edit form will submit.
    So that controller is responsible for updating a specific note.

notes/destroy.php
    destroy is responsible for destroying the note, removing it from the database
